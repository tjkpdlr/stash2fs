# stash2fs - instructions for agents

`stash2fs` is a file system organizer designed to be both a standalone application and a plugin for [stash](https://stashapp.cc/). The purpose of the tool is to organize images and videos on a filesystem according to metadata on the Stash database.

## Rules

- **always** read the `AGENTS.md` file inside each directory before making changes to files inside

- specs live under `./specs` - start with `./specs/AGENTS.md`
